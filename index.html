Work

A dice-roll game using HTML, CSS, and Javascript. The website features my own personal dice drawings and logo design. The challenges I faced while making this site revolved around the arrangement of elements. In addition, making these elements responsive took time and thinking. To solve these problems, I tried every solution I could think of until I got it right. Through this, I gained more confidence as a beginner developer. There are aspects I'd like to still improve. As I grow, I look forward to returning to this site with new knowledge.
